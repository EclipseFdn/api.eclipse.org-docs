Eclipse MarketPlace API

The Eclipse MarketPlace is a place to discover, share, and install relevant Eclipse plugins and solutions. Think of it like an app store for Eclipse solutions.

Authentication

Eclipse MarketPlace API uses OAuth Authorization with the mpc_update scope for POST and DELETE operations. Read operations do not require Authorization.

Error States

The common HTTP Response Status Codes are used.

Favorites

List of all the Eclipse MarketPlace Favorites.

Retrieve Favorites

GET

https://api.eclipse.org/marketplace/favorites{content_id}{?name,page,pagesize}
Parameters
content_id string

Unique identifier of the listing, i.e, 123, 134.
name string

A valid Eclipse username.
page string

The index of the page to get, defaults to 1.
pagesize string

The number of records to get per page, defaults to 20. Maximum value is 100.
